Big O notation is a mathematical notation used to describe the growth rate of functions. It is commonly used in computer science to analyze the time complexity of algorithms and determine how fast they perform when plotting input sizes.


1. boolean empty() – Returns whether the stack is empty
Time Complexity : O(1)
2. int size() – Returns the size of the stack
Time Complexity : O(1)
3. T peek() – Returns a reference to the topmost element of the stack
Time Complexity : O(1)
4. T push(T) – Adds the element at the top of the stack
Time Complexity : O(1)
